## Version Control
> https://github.com/leoafarias/fvm

```
npm install -g fvm
fvm 1.7.8
```
FVM 是將 ~/flutter 設定link到想要的version上，所以多個project不同version就需要做切換

IDE設定flutter path設定在 ~/flutter 上即可
